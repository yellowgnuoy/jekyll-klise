```jsx
.black-bg {
            visibility: hidden;
            opacity: 0;
            transition: all 1s;
        }
```

```jsx
.show-modal{
            visibility: visible;
            opacity: 1;
        }
```

애니메이션을 javascript에서도 넣을 수 있지만, css에서 넣는것이 더 좋다고 한다

시작 상태인 black-bg에서 끝 상태인 show-modal까지 1초에 걸쳐서 애니메이션이 진행된다.

다양한 속성으로 여러가지 애니메이션을 넣어 볼 수 있겠다.
